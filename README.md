# brace-only
Generated templates with two brace { }

## useage

```
Scope.title = 'foo'
Scope.username = 'Jimmy'
Scope.age = '23'
Scope.birth = '1994'
....
/**
*Remeber to Add $digest() method in the last
*/
Scope.$digest()
```
